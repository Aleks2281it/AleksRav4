Файлы с расширением .html открываются в браузере.

Справочник тегов.

1. Теги для текста

  • p - абзац
  • h1..h6 заголовки
  • ul/ol - списки маркированные и нумерованный li - пункт списка от англ. list item

Теги для выделения текста

  • b - жирный
  • i - курсивный
  • del - зачеркнутый

Теги для графических элементов

  • hr - линия
  • img - фотокарточка

Теги задающии структуру страницы

  • !DOCTYPE html
  • Html - содержимое веб страницы
  • head - служеб информация
  • body - содержимое html
  • header - шапка
  • nav - меню
  • main - контент веб страницы
  • section - раздел веб страницы
  • footer - подвал веб страницы
  • div - контейнер
